## 📖 What is ComfyUI_Viewer_OpenReel_Extension?

This application adds video editing features to ComfyUI Viewer by integrating OpenReel. It helps you edit videos directly in ComfyUI with an easy-to-use interface. The extension is designed for Windows users who want basic video editing without complicated setups.

## ⚙️ System Requirements

Before you start, make sure your computer meets these minimum specs:

- **Operating System:** Windows 10 or newer  
- **Processor:** Intel i3 or equivalent
